## 用户环境

* **操作系统**: Windows 11

## 交互要求

1. 你在处理所有问题时，**全程思考过程必须使用中文**（包括需求分析、逻辑拆解、方案选择、步骤推导等所有内部推理环节）；
2. 最终输出的所有回答内容（包括文字解释、代码注释、步骤说明等）**必须全部使用中文**，仅代码语法本身的英文关键词除外；
3. 在判断问题和提供解决方案时，必须结合 Windows 11 的实际环境来考虑。
4. 当用户要求记录内容的时候，除非明确说明只记录项目目录，否则要把内容记录到全局的AGENTS.md文件中

## 代码修改要求

1. **改代码要考虑性能，不新增 GC** - 避免使用 LINQ、匿名方法、闭包等会产生 GC 的模式，优先使用 for 循环、静态数组等方式
